Sitio web -Estudiante 3- Tema: Etiquetas Semánticas, Listas y Enlaces.

El propósito del sitio web para la fase 2 de la materia diseños de sitios web fomenta el aprendizaje
de los siguientes temas: Etiquetas semánticas, listas y enlaces que se verán en el transcurso del curso.

Etiquetas Semánticas

Las etiquetas semánticas permiten para que cualquier motor de búsqueda, o cualquier otro mecanismo automático
lea un sitio web y este sepa con exactitud qué partes de su contenido corresponden a cada una de las partes típicas de un sitio.
Generalmente, en cualquier documento tenemos un encabezado, un cuerpo y un pie de página, elementos que definen la estructura representados

Listas en HTML

Las listas en HTML nos permiten crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos,
generalmente, por un guion o número.
Los tipos de listas en HTML son los siguientes:
• Listas ordenadas
• Listas desordenadas
• Listas de definiciones

Enlaces en HTML

Mediante los enlaces en HTML podemos comunicar una página con otra. De esta forma, enlazando documentos HTML podemos acabar tejiendo lo que es Internet.
Para poder indicar el destino de un enlace utilizamos el [atributo href][Href]. En valor del [atributo href][Href] puede ser cualquier URI que represente un recurso.
Es decir, una página, una parte de una página, una URL genérica, un archivo.
